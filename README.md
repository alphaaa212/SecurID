## 🧩 概要

例：証明書をWeb上で一括管理できるサイト

---

## 🎯 作った目的

わざわざ財布を出して見せなければならない本人確認証。
クレジットカード番号を調べるために財布を取りに行く手間。
これらを解決するために、サイト上でサクッと証明書類等を確認できるサイトを作成しました。
